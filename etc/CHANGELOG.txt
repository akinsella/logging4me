ChangeLog:

Version 1.0.2:

- New Features:
	- XmlConfigurator support and xml configuration extensibility
- Tests added

- BUGFIXES:
	- PatternLayout always prints TRACE log level in for every log output

- Logging features optimized for J2ME & ANDROID systems
- Console Appender
- J2ME File Appender
- SimpleLayout
- Level Layout
- Pattern Layout, support custom date formatting
- Hierarchical category management
- Multi-level support (TRACE, DEBUG, INFO, WARM, ERROR, FATAL)
